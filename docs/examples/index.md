# Examples

Practical code samples showing how to use poly-data for sports prediction market analysis.

Each example is self-contained — copy-paste and run.

| Example | What you'll learn |
|---------|-------------------|
| [Fetching Events](fetching-events.md) | Query Gamma API, filter by sport, get DataFrames |
| [Price Data & Order Books](price-data.md) | CLOB midpoints, order books, price history charts |
| [ESPN Game Times](espn-game-times.md) | Match Polymarket events to real game schedules |
| [Market Filtering](market-filtering.md) | H2H filters, soccer beat/draw, esports detection |
| [Plotting](plotting.md) | Matplotlib/seaborn charts — price series, sport distributions, depth |

!!! tip "All examples use real API data"
    These examples hit live Polymarket and ESPN APIs. No API keys needed for Gamma, CLOB, Data API, or ESPN.
